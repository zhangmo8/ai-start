# Day 01

# Python 是什么
Python 是一种高级编程语言，设计目标是让代码更易读、更简洁。它有丰富的库和社区支持，适合各种应用场景，从数据分析到人工智能。

# 解释器、脚本、模块
- 解释器：Python 代码需要通过解释器来执行，CPython。
- 脚本：一个 Python 文件，可以直接运行，py __name__ == "__main__"
- 模块：一个 Python 文件，可以被其他 Python 文件导入使用。 py __name__ = "module_name"

# 变量、基本类型、表达式、条件、循环
- 变量：用来存储数据的容器，可以随时改变值。
- 基本类型：int（整数）、float（浮点数）、str（字符串）
- 复杂类型：list（列表）、dict（字典）
- 表达式：由变量、运算符和函数调用组成的代码片段，计算结果。
- 条件：if 语句，根据条件执行不同的代码块。
- 循环：for 和 while 语句，用来重复执行代码块。
