## 今天要重点看什么

你在 NumPy 里最常会碰到这些东西：

- 数组创建
- `shape`
- 索引和切片
- `reshape`
- `transpose`
- 逐元素运算
- 矩阵运算
- 广播

这些不是零散知识，它们其实都在解决同一件事：

> 数组怎么存、怎么取、怎么变形、怎么计算

---

## 怎么读文档才不容易晕

不要从头到尾一页一页死看。

用下面这个顺序更稳：

1. 先找例子
2. 看输入
3. 看输出
4. 自己改一个数字
5. 再运行确认

你要训练的是“猜结果”的能力，不是“抄标题”的能力。

---

## 你应该特别关注的关键词

### `ndarray`

NumPy 的数组对象。

### `axis`

沿着哪一维操作。

### `reshape`

改变数组的外形，不改数字本身。

### `transpose`

交换维度顺序。

### broadcasting

让不同 shape 的数组在规则下也能计算。

---

## 最小 demo

```python
import numpy as np

a = np.arange(6)
print(a)               # [0 1 2 3 4 5]
print(a.shape)         # (6,)

b = a.reshape(2, 3)
print(b)               # 2 行 3 列
print(b.shape)         # (2, 3)

c = b.transpose()
print(c.shape)         # (3, 2)
```

你可以把这个 demo 当成“看数组变形”的练习。

## 常见误区

### 误区 1：看文档就是看定义

不够。你要看例子。

### 误区 2：知道函数名就算懂了

不够。你要知道输入和输出 shape。

### 误区 3：只要能运行就算会了

不够。你得能解释为什么能运行。

---

## 你什么时候该问问题

你只要遇到下面这些情况，就应该停下来问：

- “这个例子为什么要这么写？”
- “这里为什么要 `reshape`？”
- “这个 `axis` 是什么意思？”
- “这里是切片、转置还是广播？”

可以直接问：

- “这个操作前后的 shape 分别是什么？”
- “作者想让我理解什么？”
