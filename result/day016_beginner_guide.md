# Day 016 补充讲义：向量、矩阵、张量

## 先说结论

今天只学一件事：

> 数字可以按不同层级组织起来，最简单的是向量，更复杂一点是矩阵，再往上就是张量。

你不需要先会线代公式。你只要先学会把“数字堆”看成“有结构的数组”。

---

## 这三个词到底是什么意思

### 向量 vector

向量就是一排数字。

比如：

```python
[10, 20, 30]
```

它可以表示：

- 3 个数
- 3 个特征
- 3 个方向上的信息

在 NumPy 里，它通常是 `shape=(3,)`。

### 矩阵 matrix

矩阵就是二维表格。

比如：

```text
[
  [1, 2, 3],
  [4, 5, 6]
]
```

它可以表示：

- 2 行 3 列
- 2 个样本，每个样本 3 个特征
- 2 个学生，每人 3 门成绩

在 NumPy 里，它通常是 `shape=(2, 3)`。

### 张量 tensor

张量就是更高维的数组。

比如 `shape=(2, 3, 4)`，你可以把它想成：

- 2 个大块
- 每个大块 3 行
- 每行 4 个数

所以张量不是玄学，它只是“层数更多的数组”。

---

## 你最该记住的不是名字，而是 shape

今天最重要的习惯是：

> 看到数组，先问它的 `shape`

例如：

```python
import numpy as np

a = np.array([10, 20, 30])
b = np.array([[1, 2, 3], [4, 5, 6]])
c = np.arange(24).reshape(2, 3, 4)

print(a.shape)  # (3,)
print(b.shape)  # (2, 3)
print(c.shape)  # (2, 3, 4)
```

你要练习的不是“记住这些写法”，而是把它们翻译成人话：

- `(3,)` = 一排 3 个数
- `(2, 3)` = 2 个对象，每个对象 3 个特征
- `(2, 3, 4)` = 2 组数据，每组 3 行，每行 4 个数

---

## 这一天为什么重要

因为模型代码里最常见的东西就是数组：

- 输入数据是数组
- 权重是数组
- 中间结果也是数组

如果你看不懂数组的层数，后面就会觉得模型代码像天书。

---

## 最小 demo

```python
import numpy as np

scalar = np.array(7)
vector = np.array([10, 20, 30])
matrix = np.array([[1, 2, 3], [4, 5, 6]])
tensor = np.arange(24).reshape(2, 3, 4)

print(scalar.shape)  # ()
print(vector.shape)  # (3,)
print(matrix.shape)  # (2, 3)
print(tensor.shape)  # (2, 3, 4)
```

你可以先只看 `shape`，不用管每个数是什么。

---

## 常见误区

### 误区 1：张量一定很难

不对。张量只是更高维的数组。

### 误区 2：向量就是普通列表

不完全对。列表是 Python 的容器；向量是数学和 NumPy 里的数据结构概念。

### 误区 3：矩阵只是表格名字

不完全对。矩阵后面会和乘法、变换、模型权重直接绑定。

- “这个 `shape` 应该怎么翻译成人话？”
- “这是一维、二维还是三维？”
- “每个维度代表谁？”
