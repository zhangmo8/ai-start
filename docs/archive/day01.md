# Day 01 - Python 心智模型

今天学什么：
- Python 是什么
- 解释器、脚本、模块的概念
- 变量、基本类型、表达式、条件、循环

重点理解：
- Python 是你后续读模型代码的基础语言，不是目标本身
- 先学“读懂代码”所需的部分，不追求花哨语法

去哪里学：
- Python 官方教程: https://docs.python.org/3/tutorial/
- Python 内置类型: https://docs.python.org/3/library/stdtypes.html

今天的输出：
- 写 1 页笔记，标题就叫“Python 代码长什么样”
- 总结 `int`、`float`、`str`、`list`、`dict` 各自是什么

完成标准：
- 你能看懂最基本的 Python 代码结构
- 你知道模块和脚本不是一回事
