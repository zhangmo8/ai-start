# Day 04 - 文件、JSON、CLI、日志

今天学什么：
- 文件读写
- JSON 结构
- `argparse` 的作用
- logging 的基本作用

重点理解：
- 模型工程不只是模型本身，训练配置、数据样本、日志几乎都依赖文件和 JSON
- 训练脚本没有日志，后面会很难定位问题

去哪里学：
- argparse 官方文档: https://docs.python.org/3/howto/argparse.html
- pathlib 官方文档: https://docs.python.org/3/library/pathlib.html
- logging 官方文档: https://docs.python.org/3/library/logging.html
- json 官方文档: https://docs.python.org/3/library/json.html

今天的输出：
- 写 1 页笔记，标题叫“为什么训练脚本必须有配置和日志”

完成标准：
- 你知道配置、数据、日志分别该放在哪类文件里
