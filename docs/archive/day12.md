# Day 12 - 最小字符级 Tokenizer

今天学什么：
- 字符级 tokenizer 的工作方式
- id 映射
- encode / decode 的本质

重点理解：
- 字符级 tokenizer 很弱，但最适合你建立完整链路直觉
- 从字符级开始，你更容易真正做出自己的第一个模型

去哪里学：
- Hugging Face Tokenizers Docs: https://huggingface.co/docs/tokenizers/en/index
- SentencePiece 官方仓库: https://github.com/google/sentencepiece

今天的输出：
- 写 1 页笔记，画出 `text -> token ids -> text` 的往返流程

完成标准：
- 你能说清 encode 和 decode 分别做什么
