
# 解释器

解释器，是一种程序，它可以直接编译 python 代码成字节码，并进行执行（比如 CPython），当我们编写 python 代码的时候，无需关心底层的编译和执行细节，解释器会帮我们处理这些问题。

# 脚本

本质上是一个 py 文件，里面包含了 python 代码，直接运行这个 py 文件，就会执行里面的代码，脚本通常是用来完成某个特定任务的，比如数据处理、自动化脚本等。

当 py 文件作为脚本被使用的时候，__name__ 为 "__main__"


# 模块

本质上是一个 py 文件，里面包含了 python 代码，可以直接运行这个 py 文件，但如果作为模块，通常是作为复用和拆分的模块使用，被其他脚本或者模块导入使用，脚本通常是用来完成某个特定通用任务的，比如SUM(a,b ) 这种相对来说通用，或者需要被其他模块调用的功能。

当 py 文件作为模块，__name__ 为 模块名
