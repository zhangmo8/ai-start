### 大量样本、请求或 token 需要循环处理时，怎样避免性能瓶颈或死循环？

1. 检查循环条件，确保有明确的退出条件。
2. 使用批处理或分块处理，减少每次循环的负载。
   批处理就是将样本分成小批次处理，分块处理就是将大数据分成小块逐个处理。
3. 引入超时机制，防止死循环。

### 入口逻辑和可复用逻辑如何拆分，才能让代码易测试、易维护、易复用？

入口逻辑负责"怎么进来、怎么出去"，可复用逻辑负责"进来后做什么"
1. 将入口逻辑和可复用逻辑分成不同的函数或模块。
2. 入口函数只负责调用可复用函数，并处理输入输出。
3. 可复用函数应该设计成独立的、无副作用的函数，这样更容易测试和复用。
4. 使用接口或抽象类定义可复用逻辑的规范，确保不同实现之间的一致性。
